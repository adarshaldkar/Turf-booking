# QuickCourt - Local Sports Booking Platform

## 📄 Overview
**QuickCourt** is a web platform that allows sports enthusiasts to book local sports facilities — such as badminton courts, turf grounds, and tennis tables — and create or join matches with others in their area.  
It ensures a smooth booking experience, accurate availability, and encourages community engagement through collaborative sports events.

## 🧩 Problem Statement
**💡 Title:** QuickCourt - A Local Sports Booking Platform  

**📄 Goal:**  
Build a **full-stack web application** that enables:
- Facility booking without overlaps.
- Match creation & joining.
- Role-based features for **Users**, **Facility Owners**, and **Admins**.

---

## 🌟 Core Features

### 👤 User Role
- **Authentication**
  - Sign up with name, email, password, avatar, and role.
  - OTP verification after signup.
  - Login with email and password.
- **Home Page**
  - Welcome banner/carousel.
  - Quick links to popular venues & sports.
- **Venues Page**
  - List of approved venues with search, filters, pagination.
  - Venue cards showing name, sport types, starting price, location, rating.
- **Single Venue Page**
  - Full venue details (sports, amenities, photos, reviews).
  - **Book Now** button.
- **Court Booking Page**
  - Select court & time slot.
  - View pricing and confirm booking.
  - Simulate payment flow.
- **Profile Page**
  - View/edit personal info.
  - View booking history.
- **My Bookings**
  - See booking details & status.
  - Cancel future bookings.

---

### 🏢 Facility Owner Role
- **Dashboard**
  - KPIs: Total bookings, active courts, earnings (simulated).
  - Booking calendar.
  - Charts for trends, earnings, and peak hours.
- **Facility Management**
  - Add/Edit facility details: name, location, sports, amenities, images.
- **Court Management**
  - Add/Edit/Delete courts with pricing & hours.
- **Time Slot Management**
  - Set court availability.
  - Block slots for maintenance.
- **Booking Overview**
  - View upcoming & past bookings with user info.
- **Profile**
  - View & edit personal details.

---

### 🛡️ Admin Role
- **Dashboard**
  - Global stats: total users, owners, bookings, courts.
  - Charts for booking activity, registration trends, popular sports, and simulated earnings.
- **Facility Approval**
  - View pending facility requests.
  - Approve or reject with comments.
- **User Management**
  - Search & filter users by role/status.
  - Ban/unban users.
  - View booking history.
- **Reports & Moderation (Optional)**
  - Monitor flagged facilities/users.
  - Take actions on violations.
- **Profile**
  - View/edit admin info.
